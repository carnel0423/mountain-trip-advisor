# 旅山アプリ

日本百名山100座、日本二百名山100座を対象に、登山日時、現在地、登山レベル、地域条件から候補を絞り込む静的Webアプリです。

## 主な機能

- 現在地と距離条件に応じたおすすめ表示
- 山ごとの天候、風、危険情報、アクセス情報の表示
- 噴火警戒レベル、気象庁の注意報・警報、近隣クマ目撃情報の表示
- 前日降雨後の注意、通行規制メモ、難所情報の表示
- お気に入り保存

## スマホ確認

同じWi-Fi内のスマホから確認する場合は、PowerShellで次を実行します。

```powershell
powershell -ExecutionPolicy Bypass -File .\serve-local.ps1
```

起動後に表示される `http://192.168.x.x:8080/` のようなURLをスマホで開いてください。
Windowsのファイアウォール確認が出た場合は、プライベートネットワークで許可してください。

## 直接確認

PCだけで確認する場合は [index.html](C:\Users\石渡利和\Documents\mountain-trip-advisor\index.html) を開けば使えます。

## 備考

- 天候は気象庁のエリア予報と山頂付近の座標予測を併用しています
- 通行規制は取得済みの備考と確認先リンクを表示します
- 実際の入山可否は出発前に各公式情報で最終確認してください
